# Personal Note App

A simple and elegant Android application for creating and managing personal notes. Built with Java and SQLite, this app provides an intuitive interface for users to capture and organize their thoughts.

## Features

- Create and store personal notes
- Material Design UI
- Custom toolbar with app branding
- Swipe-to-delete functionality
- SQLite database for persistent storage
- Floating Action Button (FAB) for quick note creation



### Built With
- Java
- Android SDK
- SQLite Database
- AndroidX Libraries
- Material Design Components

### Requirements
- Android Studio Arctic Fox or later
- Minimum SDK: API 21 (Android 5.0)
- Target SDK: API 34 (Android 14)

### Dependencies
```gradle
dependencies {
    implementation 'androidx.appcompat:appcompat:1.6.1'
    implementation 'com.google.android.material:material:1.11.0'
    implementation 'androidx.constraintlayout:constraintlayout:2.1.4'
}
```

## Installation

1. Clone the repository
```bash
git clone https://github.com/William306265/Personal-Note-App.git
```

2. Open the project in Android Studio

3. Build and run the application

## Usage

1. Launch the app
2. Click the + button to create a new note
3. Enter title and description for your note
4. Swipe left on any note to delete it
